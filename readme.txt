# Проект: Upwork Estimations

## Описание
Этот проект представляет собой простое веб-приложение на Flask для оценки проектов на платформе Upwork. Пользователи могут ввести информацию о своем проекте, такую как тип проекта, название, участников, сроки и конкретные требования, после чего бот предоставит текстовое описание проекта, которое можно также скачать в формате .docx.

## Структура проекта
- `app.py`: Главный файл приложения, содержащий основной код Flask.
- `templates/`: Папка с HTML шаблонами.
    - `index.html`: Главная страница HTML с формой для ввода данных о проекте.
- `static/`: Папка со статическими ресурсами.
    - `css/`: Папка со стилями CSS.
        - `style.css`: Файл стилей CSS для веб-интерфейса.

## Инструкции по установке и запуску
1. Убедитесь, что у вас установлен Python и pip.
2. Клонируйте репозиторий с проектом: `git clone <URL репозитория>`
3. Перейдите в каталог проекта: `cd upwork_project_estimations`
4. Установите зависимости: `pip install -r requirements.txt`
5. Установите переменные окружения в файле `.env`:
    - `BASE_API_URL`: URL API для общения с ботом.
    - `FLOW_ID`: Идентификатор потока для обработки запросов ботом.
6. Запустите приложение: `python app.py`

## Использование
1. После запуска приложения перейдите на главную страницу в браузере.
2. Заполните форму с информацией о проекте.
3. Нажмите кнопку "Submit" для отправки данных.
4. После обработки запроса ботом вам будет предоставлен текстовый ответ, а также ссылка для скачивания файла .docx с описанием проекта.
